멜로디 튜토리얼

똑똑하게 멜로디 사용하기

기본 명령 목록

  • !!language : 멜로디가 어떤 언어를 사용할 지 선택할 수 있어요
  • !!tutorial : 채팅창으로 멜로디 사용 설명서를 보내드립니다
  • !!help :개인 메세지로 멜로디 사용 설명서를 보내드립니다
  • !!summon : 멜로디를 보이스 채널로 호출합니다
  • !!patchnote : 무럭무럭 자라나는 멜로디의 성장 과정을 보여드립니다:D

Play 명령 목록

  • !!play {유튜브 영상 링크} : 멜로디가 당신이 입력한 링크의 영상을 연주합니다. 이미 연주 중인 영상이 있다면, 가장 마지막 순서로 연주하게 됩니다

    • 왜 3가지 방법이 있나요?

    유튜브 링크의 형태는 3가지가 있습니다
    1.단일 유튜브 주소
    2.재생목록 주소
    3.유튜브 검색

    Case1 : 단일 유튜브 주소 -> 한 곡만 재생합니다

    ex) !!play https://www.youtube.com/watch?v=MhQKe-aERsU

    Case2 : 유튜브 재생목록 주소 -> 플레이 리스트에 있는 모든 노래를 재생합니다

    ex) !!play https://www.youtube.com/watch?v=MhQKe-aERsU&list=RDMhQKe-aERsU

    Case3 : 유튜브 검색 -> 유튜브를 검색하고 맨 위에 있는 동영상을 재생합니다

    ex) !!play tangled i see the light

  • !!now : 큐에 있는 노래들을 보여줍니다
  • !!pause :멜로디가 연주를 잠시 멈추고 휴식을 취합니다
  • !!resume : 멜로디가 연주를 다시 시작합니다
  • !!skip : 멜로디가 현재 연주 중인 곡을 멈추고, 다음 곡으로 넘어갑니다
  • !!stop : 멜로디가 연주를 멈추고, 보이스 채널에서 소환 해제됩니다
  • !!now : 멜로디가 연주 중인 곡과 연주할 예정인 곡의 목록을 보여드립니다

재생목록 명령 목록

  • !playlist create {플레이리스트 이름} : 멜로디가 입력받은 이름으로 새로운 플레이리스트를 생성합니다

    • (ex) !!playlist create my_music_list

  • !!playlist add {플레이리스트 이름} {유튜브 영상 링크} : 멜로디가 플레이리스트에 입력받은 영상 링크를 저장합니다

    • (ex) !!playlist add my_music_list https://www.youtube.com/watch?v=EiVmQZwJhsA

  • !!playlist show {플레이리스트 이름} : 멜로디가 플레이리스트에 저장 된 영상 목록을 보여줍니다. 그리고 멜로디가 플레이리스트에 저장 된 모든 영상을 연주하도록 선택할 수 있습니다

    • ex) !!playlist show my_music_list

  • !!playlist load {플레이리스트 이름} : 멜로디가 보이스 채널로 소환되어, 플레이리스트에 저장 된 모든 영상을 연주합니다

    • ex) !!playlist load my_music_list

  • !!playlist search 'playlist_name' : 멜로디가 다른 사용자들의 플레이리스트를 검색합니다. 검색된 플레이리스트를 현재 재생 목록에 추가할 수 있습니다.

    • ex) !!playlist search 아이유노래모음

  • !!playlist delete {플레이리스트 이름} {영상 번호} : 멜로디가 플레이리스트의 특정 곡을 리스트에서 제거합니다. 아래 예제에서는 iu 플레이리스트의 2번째 곡을 제거합니다

    • ex) !!playlist delete iu 2

  • !!my : 내가 만든 모든 재생목록을 보여줍니다
