Melody Tutorial

How to use Melody bot wise

list of basic command

  • !!language :You can choose which language Melody will use
  • !!tutorial : Melody sends 'Melody User's Guide' on chat
  • !!help : Melody sends 'Melody User's Guide' by direct message
  • !!summon : summon melody to my voice channel
  • !!patchnote : Show patch notes for this version

List of play command

  • !!play 'youtubelink' : Melody plays the video of the link you entered. If Melody is already playing something, plays the video at the end of the queue.

    • Why there is 2 case?

    There is two type of youtube link
    1.sigle video link
    2.playlist link.
    3. text to search in youtube

    Case1 : single video link -> play single song

    ex) !!play https://www.youtube.com/watch?v=MhQKe-aERsU

    Case2 : youtube playlist of video -> play all song in playlist

    ex) !!play https://www.youtube.com/watch?v=MhQKe-aERsU&list=RDMhQKe-aERsU

    Case3 : text to search on youtube -> search text in youtube and play top listed video

    ex) !!play tangled i see the light

  • !!pause :Melody pauses playing and rests
  • !!resume : Melody resumes playing
  • !!skip : Melody stops playing and skips to next
  • !!stop : Melody stops playing and leaves from the voice channel
  • !!now : Melody shows the list of songs that is playing and is about to play

List of playlist command

  • !!playlist create {playlist name} : Melody creates a new playlist with the name you entered

    • (ex) !!playlist create my_music_list

  • !!playlist add {playlist name} {youtube link} : Melody saves the video link to the playlist

    • (ex) !!playlist add my_music_list link

  • !!playlist show {playlist name} : Melody shows a list of videos stored in the playlist. And you can choose whether or not to play all the videos stored in the playlist

    • (ex) !!playlist show my_music_list

  • !!playlist load {playlist name} :Melody is summoned to the voice channel and plays all the videos stored in the playlist

    • (ex) !!playlist load my_music_list

  • !!playlist search 'playlist_name' : Melody searches playlist of other users. You can add searched playlist to current playing.

    • ex) !!playlist search IUSongs

  • !!playlist delete {playlist name} {number} : Delete song with index number

    • ex) !!playlist delete iu 2 -> delete second 2 song in iu playlist

  • !!my : show all my playlist
